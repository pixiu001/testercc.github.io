Java

后端老大哥地位不可动摇。

学习推荐

Java秘籍一本 -- 每天半小时到一小时， 一年精通Java。

java打包

命令行打包名为pctest的项目

jar -cvf 1.jar .

java -jar 1.jar    # 如果提示没有主属性，可以用解压软件打开1.jar文件，修改META-INF/MANIFEST.MF内容，指定主类。比如加上: Main-Class: com.company.Main

jvm性能调优工具

jps   类似于linux的ps命令,但是它只列出系统中所有的Java应用程序
