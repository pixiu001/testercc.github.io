Vim 快捷操作

一般来说，线上开发环境大家都习惯用命令行修改，故基本的Vim快捷操作还是要熟悉的。

一般模式

按小写u键为撤销上一步操作

按ctrl+r则为反撤销操作

Vim 批量注释

方法1（常用）

1.ctrl+v 进入列编辑模式，向下或向上移动光标，把需要注释的行的开头标记起来。
2.按大写的I(shift+i),再插入注释符,比如"//"。
3.再按Esc,就会在选中的行前面都插入"//"，即全部注释了。

方法2 (利用全局替换)

行首添加注释

使用下面命令在指定的行首添加注释：

:起始行号,结束行号s/^/注释符/g

取消注释

:起始行号,结束行号s/^注释符//g

1.在10 - 20行添加 // 注释

:10,50s#^#//#g

2.在10 - 20行删除 // 注释

:10,20s#^//##g
