SEO相关术语

一般而言，学习一门新技术时，首先要了解一些常见术语，否则，即使找到不错的学习资料，也会看得云里雾里的。

SEO (Search Engine Optimization)

学名"搜索引擎优化"，是一种方式：利用搜索引擎的规则提高网站在有关搜索引擎内的自然排名。
  • mkdocs new [dir-name] - Create a new project.
  • mkdocs serve - Start the live-reloading docs server.

SEM (Search Engine Marketin)

学名"搜索引擎营销"，就是根据用户使用搜索引擎的方式利用用户检索信息的机会尽可能将营销信息传递给目标用户。企业通过搜索引擎付费推广，让用户可以直接与公司客服进行交流、了解，实现交易，也就是俗称的“竞价推广”。其中最典型最著名的就是百度SEM（应该算百度盈利最多的业务之一了，当然也给百度带来很多负面影响），本人曾经测试过，有些热点关键词在某些时段甚至会使第一页全部展现成有“广告”标识的搜索结果。

因为此笔记以讲解SEO为主，故这里仅简单介绍SEM定义，以便区分。实际运营团队中，竞价专员（专职做SEM）和SEO专业的分工还是很明确的。
