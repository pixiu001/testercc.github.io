  • 机器学习

因为一直希望能进行安全业务和机器学习技术相关的研发，所以机器学习相关知识肯定已加入学习队列。鉴于机器学习知识覆盖范围和深度都极广，所以这里仅记录个人的一些碎片化笔记，仅供参考。

阅读推荐

看这基本书之前最好具备以下基础：

0.基础的python编程
1.线性代数，微积分和概率统计，不用复习的很深入，基本的原理懂和例题会做即可
  • 吴恩达的机器学习视频 -- 初始入门推荐学习，对整个机器学习流程和体系有个大概了解，重要的是多跑代码，多写。
  • 《统计学习方法》 李航 -- 第一版很出明，现在已经有第二版了。
  • lihang_book_algorithm -- 这个作者将《统计学习方法》中每一章的算法用python实现一遍，得到了李航老师的认可。 建议配套《统计学习方法》书籍，边看书边手推公式然后对着GitHub一边敲代码。
  • 《机器学习》 周志华 -- 江湖人称"西瓜书" -- 身边学习机器学习的小伙伴基本人手一本（纸质/电子版）。
  • pumpkin-book《南瓜书》 Github Repo -- 针对西瓜书部分公式推倒太难的问题，GitHub上有个开源项目，由开源组织Datawhale发起，得到了周志华老师的认同，叫做pumpkin-book《南瓜书》
  • 《机器学习实战》Peter Harrington -- 代码地址 每一章都用代码实现了一部分机器学习的算法，对理解算法的原理很有帮助。
